La solution

Un réseau IA comme BitTorrent,
mais pour votre cerveau.

Imaginez un village où chacun prête un peu de sa puissance de calcul aux autres. En échange, quand vous avez besoin d'IA, le village travaille pour vous.

1. Vous rejoignez

Une seule commande. Votre machine reçoit une identité cryptographique unique. Pas de nom, pas de mail, pas de mot de passe.

2. Vous contribuez

Quand vous n'utilisez pas votre ordinateur, il prête sa puissance aux autres. Vous accumulez du crédit de calcul.

3. Vous utilisez l'IA

Le réseau distribue votre requête sur plusieurs machines. Vous recevez la réponse en temps réel.

Pourquoi AInonymous

Tout ce qu'on vous a retiré,
on vous le rend.

🔐

Confidentialité absolue

Installation

Prêt en 30 secondes.

# 1. Cloner le dépôt
git clone https://github.com/Geoking2104/AInonymous.git
cd AInonymous

# 2. Builder
cargo build --release

# 3. Lancer le daemon
./target/release/ainonymous-daemon
