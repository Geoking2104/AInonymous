Architecture avancée

HybridNode : une architecture à trois couches

AInonymous ne se limite pas à un simple réseau P2P. HybridNode combine SD-WAN, Holochain et QUIC/mTLS pour distribuer l'inférence en tenant compte de la topologie réseau.

CoucheTechnologieRôle
OverlayHolochain 0.6.1DHT, identité (AgentPubKey ed25519), coordination
Plan de donnéesQUIC / mTLSTransfert des activations tensorielles, flux de tokens
UnderlaySD-WANRoutage conscient de la topologie, SLA, QoS
🔒

mTLS strict

La clé AgentPubKey ed25519 sert de certificat QUIC ; PeerKeyVerifier impose l'authentification mutuelle.

🧬

Attestation de modèle

Vérification SHA-256 locale, confirmée par au moins 2 pairs du réseau.

⚠️

Système de mandats (warrants)

Preuve cryptographique de mauvais comportement stockée dans le DHT Holochain ; exclusion automatique du scheduling.

🔐

Bootstrap privé

Membrane PrivateNetworkProof pour les déploiements en consortium fermé.

# Installer depuis les sources
cargo install --path crates/hybridnode-daemon --features mock-sdwan

# Initialiser la config d'un nouveau projet
bash scripts/hybridnode/init_project.sh mon-projet

# Lancer
hybridnode --config hybridnode/configs/mon-projet.hybridnode.yaml

Documentation complète : docs/HYBRIDNODE.md sur GitHub.

Advanced architecture

HybridNode: a three-layer architecture

AInonymous isn't just another P2P network. HybridNode combines SD-WAN, Holochain, and QUIC/mTLS to schedule inference with full network-topology awareness.

LayerTechnologyRole
OverlayHolochain 0.6.1DHT, identity (AgentPubKey ed25519), coordination
Data planeQUIC / mTLSTensor activation transfers, token streams
UnderlaySD-WANTopology-aware routing, SLA enforcement, QoS
🔒

Strict mTLS

The ed25519 AgentPubKey is reused as the QUIC certificate; PeerKeyVerifier enforces mutual authentication.

🧬

Model attestation

SHA-256 verified locally, confirmed by at least 2 peers on the network.

⚠️

Warrant system

Holochain DHT cryptographic proof of misbehavior; automatically excludes the peer from scheduling.

🔐

Private bootstrap

PrivateNetworkProof membrane for closed consortium deployments.

# Install from source
cargo install --path crates/hybridnode-daemon --features mock-sdwan

# Initialize a new project config
bash scripts/hybridnode/init_project.sh myproject

# Run
hybridnode --config hybridnode/configs/myproject.hybridnode.yaml

Full documentation: docs/HYBRIDNODE.md on GitHub.
