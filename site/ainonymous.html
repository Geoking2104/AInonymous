Comment ça marche

Vous rejoignez le réseau en 30 secondes. Votre machine prête sa puissance quand elle est inactive. Quand vous avez besoin d'IA, le mesh distribue le calcul sur plusieurs nœuds via pipeline-splitting. Vous recevez la réponse en temps réel. Aucun serveur central n'a jamais vu votre prompt.

1. Rejoindre
Une commande installe tout. Identité cryptographique ed25519 automatique.
2. Contribuer
GPU/CPU idle = crédit de calcul pour la communauté (troc de ressources).
3. Utiliser
API OpenAI-compatible en local. Le mesh s'occupe du reste.
