Installation

Ready in 30 seconds.

One command. Your machine joins the network, picks a model suited to its power, and starts contributing — and receiving.

# 1. Install AInonymous
curl -fsSL https://ainonymous.network/install.sh | sh

# 2. Join the network automatically
ainonymous --auto

# 3. Or pick your model
ainonymous --model gemma4-26b-moe

# 4. Test in 5 seconds
curl http://localhost:9337/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"model":"gemma4-31b","messages":[{"role":"user","content":"Hello!"}]}'
